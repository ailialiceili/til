# AIを使った開発の進め方

AI開発には作法がある。

## 手順

### 1. 要件定義

- 目標を決める（例：Todoアプリを作りたい）
- まずは要件定義から始める
- 不明点をつぶすために、わからないところをAIに質問する

**例：環境・技術スタックの明示**
- Windows で動く
- Python 開発

- AIが返してきた質問をさらにつぶしていく

### 2. 要件定義書の作成

AIへの最終プロンプト例：

```
要件定義書として保存したいので不要な文章は全部削除
最初と最後の挨拶は不要
純粋な要件定義書の文書として出力してください
```

AIの応答をコピーして、リポジトリ内の `docs/requirements.md` にマークダウン形式で保存する。

### 3. 開発開始

AIへのプロンプト例：

```
@/docs/requirements.md
この要件定義書の内容を基に開発を開始してください
pip install 必要なものは /requirements.txt にまとめてください
```

## Tips

- 新しいことをやるときは**新しいチャット**を始める
